# HTML5 语义化与 SEO 基础

语义化不是“写对标签”这么简单，而是让搜索引擎和辅助技术准确理解页面结构。

## 1. 标题层级

每个页面应只有一个主标题（`h1`），并按 `h2`、`h3` 逐层展开，不要跳级。

## 2. 核心语义标签

- `header`：页头或区块头部。
- `main`：页面主内容（通常只出现一次）。
- `section`：主题分组内容。
- `article`：可独立分发的完整内容。
- `footer`：页脚或区块脚注。

## 3. 基础 SEO 必做项

- 为每个页面设置唯一的 `title`。
- 提供简洁的 `meta description`。
- 图片加有意义的 `alt`。
- 避免死链与重复内容。

## 4. 质量基线

- 页面在无 JavaScript 情况下应仍可浏览主内容。
- 表单字段必须有可关联的 `label`。
- 外链新开窗口时加 `rel="noopener noreferrer"`。
